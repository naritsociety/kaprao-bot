## 🚀 ภาพรวมของระบบ (System Overview)

แพลตฟอร์มนี้เป็นระบบ **Multi-tenant Ready** ที่ออกแบบโครงสร้างฐานข้อมูลและการส่งท่อข้อมูลให้รองรับการแยกแบรนด์และข้อมูลออกจากกันชัดเจนด้วย `shop_id` ตั้งแต่ Commit แรก เพื่อรองรับการขยายตัวสู่ระบบ SaaS ในอนาคต ปัจจุบันรองรับการทำงาน 2 แบรนด์หลัก:

1. **กะเพราวันหยุด (`shop_id: 1`)** - เมนูกะเพราโบราณคั่วแห้งพรีเมียม ถ่ายภาพ Moody & Tone ด้วย Canon 750D เจาะกลุ่มลูกค้า High-end
2. **ร้านข้าวไข่เจียว (`shop_id: 2`)** - แบรนด์ออนไลน์แยกต่างหากเพื่อเสิร์ฟเมนูแมส เข้าถึงง่าย โดยไม่ทำลาย Signature ของแบรนด์หลัก

## 🛠️ เทคโนโลยีที่ใช้ (Tech Stack สายซิ่ง)

- **Runtime:** `Bun` (เครื่องยนต์สายซิ่ง ยุค 2026 เร็วแรงกว่า Node.js เพียวๆ)
- **Back-end Framework:** `ElysiaJS` / `TypeScript` (ตรรกะเป๊ะ Type-safe ถือตะหลิวมั่นใจ)
- **Front-end Ready:** `Next.js` (สำหรับหน้าระบบสั่งอาหาร และ Dashboard สรุปกำไร 50%++)
- **Web Server & Reverse Proxy:** `Nginx` (คุมท่อส่งน้ำรับ HTTPS พ่นเข้าพอร์ตภายใน)
- **Process Manager:** `PM2` (รันบน Fork Mode ครอบด้วยสคริปต์ฟื้นคืนชีพอัตโนมัติเมื่อ Server Reboot)
- **Database:** `MariaDB` (จัดเก็บโครงสร้างข้อมูลออเดอร์, ประวัติลูกค้า และสถิติยอดขาย)
- **CI/CD:** `GitHub Actions` (วิถีเทพ Push ไม้ต่อจากออฟฟิศถึงคอนโด สั่ง Deploy อัตโนมัติลง DC)

## 💻 คำสั่งสำคัญสำหรับนักพัฒนา (Developer Command Cheat Sheet)

### 1️⃣ ลูปอมตะส่งไม้ต่อในการพัฒนา (Git Flow)

เมื่อต้องการย้ายสถานที่ทำงาน (ออฟฟิศ <-> คอนโด) บน branch `main`:

```bash
# ก่อนเริ่มเขียนโค้ดทุกครั้ง ดึงของล่าสุดลงมาฟาด
git pull origin main

# หลังเขียนโค้ดเสร็จ/เตรียมย้ายที่ ดันขึ้นคลาวด์
git add .
git commit -m "feat: พัฒนาลอจิกบอทเพิ่มเติม จ๊วดๆ"
git push origin main
```

### 2️⃣ การควบคุม Process หลังบ้านบนเซิร์ฟเวอร์ (PM2 Control)

```bash
# ตรวจสอบสถานะความจ๊วดของบอท
pm2 list

# สั่งหยุดและลบผีตัวเก่าออกเพื่อเคลียร์ Memory
pm2 delete kaprao-api

# สั่งรันขึ้นมาใหม่ด้วยสิทธิ์จำลองลิงก์ Node -> Bun
pm2 start index.ts --name "kaprao-api" --interpreter node

# บันทึกสถานะปัจจุบันกันเหนียวเวลา Server Reboot
pm2 save
```

### 3️⃣ การจัดการคลังรูปภาพเสิร์ฟเมนู (Nginx Static Files)

คลังรูปภาพอาหาร Moody ยั่วๆ จากกล้อง Canon 750D จะถูกจัดเก็บและเข้าถึงผ่านทาง:

| ประเภท             | เส้นทาง                                 |
| ------------------ | --------------------------------------- |
| **Path ในเครื่อง** | `~/kaprao-bot/public/images/`           |
| **URL สาธารณะ**    | `https://api.weekendkaprao.com/images/` |
